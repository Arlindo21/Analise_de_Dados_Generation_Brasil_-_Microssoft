Curso de Análise de Dados: Python, IA, Excel e BI
Generation Brasil
Neste repositório estarão alocados alguns arquivos e projetos desenvolvidos no dia a dia durante o programa. O curso foca na intersecção entre finanças, tecnologia e análise de dados, preparando para desafios reais de Business Intelligence e Inteligência Competitiva.

🚀 Tecnologias e Ferramentas Utilizadas
Linguagens e Bibliotecas (Python)
Para o processamento e visualização de dados, utilizamos as principais bibliotecas do ecossistema Python:

Pandas & NumPy: Manipulação, limpeza e tratamento de grandes volumes de dados.

Matplotlib & Seaborn: Criação de gráficos estatísticos e visualizações detalhadas.

Scikit-Learn: Implementação de modelos iniciais de Machine Learning e análise preditiva.

Requests: Integração com APIs para coleta de dados externos.

Ambiente de Desenvolvimento
Jupyter Notebook (.ipynb): Utilizado para documentar todo o processo de análise de forma interativa, unindo código, visualizações e textos explicativos em um único documento.

Ferramentas de BI e Produtividade
Power BI: Construção de Dashboards interativos e modelagem de dados.

Excel Avançado: Automação com fórmulas complexas e estruturação de dados.

SQL: Extração e manipulação de dados em bancos relacionais.

📂 Estrutura do Repositório
O conteúdo está organizado para facilitar a navegação:

/Notebooks: Arquivos .ipynb com análises exploratórias e projetos de IA.

/Scripts: Arquivos Python (.py) voltados para automação.

/Datasets: Bases de dados utilizadas nos exercícios (respeitando a privacidade de dados).

/Excel_BI: Dashboards e planilhas de análise financeira e operacional.
